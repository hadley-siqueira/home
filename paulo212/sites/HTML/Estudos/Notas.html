Terminal

cd

Serve para mudar de diretorio.

pwd

Mostra o diretorio atual que estamos.

ls

Exibe os conteudos do diretorio atual, mas se você passar um diretorio como argumento, ele ira mostrar os arquvos e pastas de tal diretorio. pode ser usado de forma diferente com a flag -l e -lh, onde o -l faz o ls exibir linha por linha e com mais detalhes e o -lh faz com que os tamanhos dos arquivos e pastas sejam exibidos.

mkdir

O mkdir serve para criar novas pastas.

cat

O cat serve para verificar rapidamente o conteudo de um arquivo sem precisar abrir o mesmo no vim, As vezes, o arquivo й grande demais e nгo cabe na tela. Para isso, usamos o comando cat associado com o comando less.

mv

O mv recebe como argumento o arquivo/pasta a ser movido e o diretorio destino.

rm -rf

deletar arquivos e pastas. Usamos o comando rm -rf para excluir o que queremos.

cp

copiar arquivos e pastas usando o comando cp Voce deve especificar o arquivo/pasta a ser copiado e o diretorio de destino, Quando vamos copiar uma pasta com arquivos dentro, usamos a flag -R.
