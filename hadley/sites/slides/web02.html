Os três pilares

Frontend

Os três pilares do desenvolvimento frontend são: HTML5, CSS e Javascript

HTML

HTML é a linguagem base para o desenvolvimento de um site. É nela que se insere os textos, imagens, vídeos, audios etc que serão vistos e ouvidos pelo usuário.

CSS

CSS é a linguagem usada para customizar uma página HTML. Sua função é especificar cores, fontes, posicionamentos, etc de uma página HTML

Javascript

Javascript é a linguagem de programação disponível para o desenvolvimento frontend. Sua principal função é permitir páginas interativas e dinâmicas

Fim da aula

HTML é a linguagem base para o desenvolvimento de um site. Sua versão mais atual é conhecida por HTML5

Foi desenvolvida por Tim Berners Lee em 1980 e ganhou força em 1990. Seu objetivo era o compartilhamento de informações tecnicas pela internet

HTML é uma linguagem de marcação e não de programação. Usa o conceito de tags para marcar o texto.

As tags no HTML servem para marcar um texto. Um texto pode ser marcado como sendo:

  • Um cabeçalho
  • Um parágrafo
  • Uma lista
  • Uma tabela
  • Etc

Uma tag é um código HTML que começa com < e termina com >

Algumas tags trabalham em pares enquanto outras trabalham sozinhas. Exemplos de algumas tags:

  • <p></p> para marcar o texto como um parágrafo
  • <h1></h1> para marcar o texto como um cabeçalho
  • <br> para quebra de linha

Criando a Primeira página HTML

Arquivos HTML são simples arquivos de texto com extensão .html

Um arquivo HTML é divido em cabeça e corpo. A cabeça é responsável por inclusões de outros arquivos e configurações da página. O corpo contém o conteúdo que será exibido para o usuário.
