Introdução ao CSS

CSS (Cascading Style Sheet) é a linguagem usada para customizar uma página HTML

Tem uma sintaxe simples: escolhe-se o elemento que se quer customizar e aplica-se as customizações

CSS deve ser incluído na cabeça do arquivo HTML, seja embutido ou como um arquivo externo

Cores

Primeiro exemplo de uso de CSS: customizar as cores

Mas antes do exemplo: entender como as cores funcionam

Qualquer cor pode ser gerada a partir da combinação das cores (ou canais) vermelho (red), verde (green) e azul (blue).

No computador, os canais geralmente variam de 0 à 255 (em decimal) ou de 0 à FF (em hexadecimal)
